## Getting Started

Install the module with: `npm install md5-google`

```js
var md5-google = require('md5-google');
md5-google.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g md5-google
$ md5-google --help
$ md5-google --version
```
